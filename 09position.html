css position 관련

개요

지금까지는 margin/padding 도는 인라인/블록 요소의 특성에 의해 부분적으로나 요소의 위치를 지정할 수 있었음반면 position이라는 속성을 이용하면 html요소들의 위치를 좌표등으로 결정할 수 있음즉 top/left/right/bottom 등의 속성으로 요소의 위치를 정할 수 있음

static position

기본적인 요소 배치순서에 다라 위>아래 왼>오른쪽으로 배치되며 부모 자식 관걔의 요소는 부모요소에 따라 배치됨

box#2
box#3

absolute position

조상요소(body) 또는 부모요소를 기준으로 top/left/right/bottom만킄 요소를 이동시켜 배치 단 부모 요소를 기준으로 하위요소를 배치하며면 부모요소의 position은 relative이어야 함
